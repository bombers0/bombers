## Логика

### Кнопка "Играть"

- Пользователю подбирается комната по его рейтингу;
- Если свободных комнат нет, то игрок занимает очередь.

### Кнопка "Войти"

- Пользователь подключается к комнате;
- Кнопка становится неактивной, если все игровые слоты заняты.
